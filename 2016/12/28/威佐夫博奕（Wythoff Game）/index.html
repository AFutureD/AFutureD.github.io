51nod 威佐夫博奕（Wythoff Game）

发表于 2016-12-28 | 更新于 2018-07-03 | 分类于 Algorithm

类型：Wythoff Game

描述：

​ 有2堆石子。A，B两个人轮流拿，A先拿。每次可以从一堆中取任意个或从2堆中取相同数量的石子，但不可不取。拿到最后1颗石子的人获胜。假设A，B都非常聪明，拿石子的过程中不会出现失误。给出2堆石子的数量，问最后谁能赢得比赛。
